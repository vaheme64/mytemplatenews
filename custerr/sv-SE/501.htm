Serverfel

501 - Värden i huvudet anger metoder som inte är implementerade.

Sidan som du söker kan inte visas, eftersom huvudet i begäran innehåller ett värde som inte överensstämmer med konfigurationsinställningarna på webbservern. Det kan till exempel vara ett värde för POST till en statisk fil som inte kan ta emot data, eller ett Transfer-Encoding-värde som inte kan använda komprimering.
